# Calaverita - Ofrenda Virtual

Una aplicación interactiva de ofrenda virtual para el Día de Muertos con animaciones, música y elementos decorativos.

## Requisitos

- Python 3.x
- tkinter (incluido con Python)
- Pillow
- pygame

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/LuisMatla/calavera.git
cd calavera
```

2. Instala las dependencias:
```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python3 calaverita.py
```

## Características

- Animaciones de calaveras (GIFs)
- Música de fondo
- Velitas animadas
- Papel picado decorativo
- Texto de calaverita con animación
- Cursor personalizado de calavera
- Imágenes de tumbitas
